## 1. Paleta de Cores – Aplicar Globalmente

### Cores Definidas

| Nome | Hexadecimal |
|------|-------------|
| Verde CTA | `#0F9D58` |
| Verde Hover | `#0C7A45` |
| Azul Principal | `#1E40AF` |
| Azul Secundário | ` #3B82F6` |
| Fundo | `#F1F5F9` |
| Texto Principal | `#0F172A` |

### Regras de Aplicação

- **CTA sólido verde**: botões principais usar `#0F9D58`, hover em `#0C7A45`
- **Ícones e elementos de IA**: usar azul (`#1E40AF` ou `#3B82F6`)
- **Métricas com destaque azul**
- **Fundo claro neutro**: `#F1F5F9` para áreas gerais
- **Texto principal**: `#0F172A`
